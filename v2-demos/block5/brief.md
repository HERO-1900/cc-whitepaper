### 0.2 节奏纪律：呼吸位 = 主动降温

Block 5 是 Pentagram 式"震撼 → 降温 → 推进"的**降温拍**。这意味着：

- **字号降一档**：本块的"最大字号" 必须**显著小于** Block 4 的 `--cc-font-size-6xl`（建议本块最大字号 `--cc-font-size-4xl` ~ `5xl`，仅用在 Q2 对照卡）
- **动画密度降一档**：Block 4 的 countUp + 多元素入场 → Block 5 仅需要必要的 fadeIn / sticky-on-scroll
- **数字密度降一档**：Block 4 是 5 hero 巨数 + 6 次级 stat → Block 5 仅 4 个白皮书规模微数字 + 5 bullet 内嵌数字

## 这本书要做什么

本书基于 Claude Code **2.1.88 版本的完整源代码**进行逆向解剖（源码分析截止日期：**2026 年 4 月**，Claude Code 后续版本的变更不在本书覆盖范围内）。我们不猜测，不臆断——所有结论都标注了源文件路径和行号。

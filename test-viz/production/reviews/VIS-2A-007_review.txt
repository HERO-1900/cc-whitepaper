## 缺陷列表

### 布局规整性

1. **grad3 颜色值错误** (SVG `<defs>` 第6个 gradient)
   - 值 `#6bbbb` 是3位缩写但只有4位字符，应为 `#6bbbbb`

2. **Layer 2 与 Layer 1 间隙过小** (SVG y坐标)
   - Layer 2 结束于 y=335，Layer 1 开始于 y=340，间隙仅5px
   - 与 Layer 3 → Layer 2 之间25px间隙明显不均

3. **Cache boundary 虚线框遮挡 Layer 2** (SVG y=141~169)
   - 缓存边界矩形 y=-14, height=28 → 绝对位置 y=141~169
   - Layer 2 起始于 y=160，该矩形上半部分(141~169)压在 Layer 2 内部

4. **scope: org 括号完全未对齐** (SVG x=680, y=195~270)
   - 括号覆盖 y=195~270，但对应的 Layer 3 (y=95~155) 在其上方40px处
   - Layer 2 (y=160~335) 的顶部在 org 括号内部

5. **scope: global 括号高度不足** (SVG x=680, y=0~190)
   - Layer 2 起始于 y=160，但 global 括号 y=0~190 仅覆盖 Layer 2 上部30px
   - 意图标注的 "global 跨组织可缓存" 应涵盖 Layer 1+2

6. **STABILITY 标签位置错误** (SVG y=336)
   - "STABILITY ↓" 组 transform=translate(120, 320)，文字 y=336
   - 绝对位置 y=336 落入 Layer 1 范围(y=340~425)内部，而非两层之间

7. **VOLATILITY 标签位置偏向下** (SVG y=70)
   - 绝对位置 y=70，落入 Layer 4 范围内(y=60~92)，而非 Layer 5~6 区域

### 视觉美学

8. **Layer 6 高度仅17px** (SVG y=0, h=17)
   - 内容拥挤，文字字号11px行高紧凑

9. **scope 括号透明度0.4过淡** (CSS `.cache-boundary` 动画)

10. **Emoji 作为 icon**

## 修改建议

1. **修复 grad3**: `#6bbbb` → `#6bbbbb`

2. **调整 Layer 1 起始位置**: 从 y=340 改为 y=335，减小与 Layer 2 的视觉割裂

3. **Cache boundary 矩形上移**: 当前 y=-14 改为 y=-20 (绝对位置 y=135)，避免遮挡 Layer 2 顶部

4. **重排 scope 括号垂直位置**:
   - global: y=-10, h=200 (覆盖 Layer 1+2)
   - org: y=165, h=120 (覆盖 Layer 3+4)
   - null: y=295, h=130 (覆盖 Layer 5+6)

5. **STABILITY 标签上移**: transform 从 y=320 改为 y=300，文字绝对位置约 y=315，位于 Layer 1~2 之间

6. **VOLATILITY 标签上移**: transform 从 y=60 改为 y=40，文字绝对位置约 y=55，位于 Layer 5~6 区域

7. **Layer 6 高度从17增至22px**: 改善可读性

8. **将 Emoji 替换为 SVG icon**: 使用 Lucide/Feather 等统一风格的 SVG 图标

9. **scope 括号透明度 0.4 → 0.6**: 提升可见性

10. **调整各层层间间距均匀化**: 当前间歇为 25px, 5px, 5px, 3px, 3px，建议统一为 8-10px

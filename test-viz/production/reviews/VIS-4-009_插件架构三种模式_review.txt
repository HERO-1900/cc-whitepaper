## 修改建议

1. **修复 Hooks SVG marker 顺序**：将第二个 `<defs><marker id="arrow">...</marker></defs>` 整体移至第一个 `<defs>` 块内（`linearGradient` 旁边），确保 `id="arrow"` 在 `marker-end` 引用前已定义

2. **删除 Script Executor 的 marker-end**：矩形闭合路径不应使用 `marker-end`，改为在矩形上方或下方用 `<line>` 元素绘制带箭头的指示线

3. **将分隔线改为 CSS pseudo-element 或 grid gap 背景色**：当前绝对定位分隔线易错位，可改用 `column::after` + CSS 渐变实现，永久消除偏移风险

4. **降低 noise 透明度至 0.02**：`opacity: 0.04` → `0.02`，或调高 `baseFrequency` 至 `1.2` 使颗粒更细密柔和

5. **精简信息层次**：表格仅保留维度对比（复杂度、安全性、能力范围），删除与卡片重复的"典型用例"行，或将表格改为纯特征矩阵（仅 Complexity / Security / Capability 三维）

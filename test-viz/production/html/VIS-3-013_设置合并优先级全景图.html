Merge Semantics · 合并语义
CROSS-SOURCE 跨来源合并
不同来源间的合并使用深度合并策略：
• 数组 → 拼接去重 (uniq)
• 对象 → 深度递归合并
• 其他 → 后者覆盖前者
settingsMergeCustomizer()
lodash mergeWith()
SAME-SOURCE 同来源更新
同一来源的配置更新使用替换策略：
• undefined → 删除键
• 数组 → 完全替换
• 对象 → 默认 merge 行为
updateSettingsForSource()
Anti-Mutation: clone()

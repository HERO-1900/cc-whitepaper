Defense in Depth Swiss Cheese Model

多层防线的设计哲学 · 纵深防御策略

A

恶意命令 / Malicious Bash

攻击尝试执行恶意脚本 → 被第2层沙箱 + 第3层权限联合拦截

B

数据外泄 / Data Exfiltration

敏感数据试图外传 → 被第1层企业策略 + 第2层网络隔离阻断

C

提权攻击 / Privilege Escalation

尝试获取更高权限 → 被第3层权限链 + 第4层代码验证阻止

"纵深防御" ≠ "偏执" — Defense in Depth ≠ Paranoia

任何单层都有漏洞，但5层组合使穿透概率趋近于零

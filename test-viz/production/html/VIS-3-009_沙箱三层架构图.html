Sandbox Architecture

Three-Layer Security System

Tool Layer 决策层

  • shouldUseSandbox() 二元决策
  • 检查是否启用沙箱
  • 处理 excludedCommands 匹配
  • 三种匹配模式：精确/前缀/通配符
  • ⚠ 警告：这不是安全边界

Adapter Layer 配置转换层

  • sandbox-adapter.ts (985行核心)
  • convertToSandboxRuntimeConfig()
  • ISandboxManager 接口定义
  • initialize() / wrapWithSandbox() / refreshConfig()
  • 企业策略覆盖: allowManaged*Only
  • 路径相对于设置文件解析

Runtime Layer 执行引擎层

  • @anthropic-ai/sandbox-runtime
  • macOS: seatbelt profile生成
  • Linux: bubblewrap + seccomp
  • Unix socket路径过滤(macOS)
  • 支持未来Landlock替换

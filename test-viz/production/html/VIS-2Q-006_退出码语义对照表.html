Unix Convention · 设计渊源
Claude Code 的退出码设计借鉴了 Unix 惯例：exit 0 表示成功，exit 1 表示命令自身错误，而 exit 2 专用于"用法错误 / 策略拒绝"（misuse / policy denial）。这正是 exit 2 作为"阻断"信号的反直觉之处——它不是普通错误，而是一个有意识的政策拦截。
0 Success · 成功完成
1 General error · 命令自身错误
2 Misuse / Policy denial · 用法错误或策略拒绝

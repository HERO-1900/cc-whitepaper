Multi-Agent Orchestration · Claude Code 不是一个 AI 在干活，而是一个 AI 团队

Claude Code 不是单线程 AI，而是支持多 Agent 协作。把它想象成一家公司：Coordinator (CEO) 接到用户任务后，派给 3 个部门经理 (Worker A/B/C)，Worker B 还可以再派给员工 (Sub-Worker)。3 种 Worker 隔离级别从弱到强：共享内存 < Git Worktree < 远程进程。所有 Agent 通过底部的 Scratchpad (共享白板) 同步进度，互相知道彼此在做什么。

读懂这张图必须知道的 5 个词

术语大白话解释类比
Coordinator主 Agent，接收用户任务后负责任务分发公司 CEO
Worker子 Agent，接受 Coordinator 派发的子任务部门经理 / 员工
WorktreeGit 的独立工作目录，让每个 Worker 有自己一份代码副本每个人一份打印资料，互不污染
Scratchpad共享内存空间，Agent 们读写状态用的中间区办公室白板，大家都能看
IPCInter-Process Communication，进程间通信同事之间用对讲机说话

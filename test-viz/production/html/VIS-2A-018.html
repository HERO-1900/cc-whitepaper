通信可靠性

向下 = 可靠（直接调用）
向上 = 可靠（XML注入）
平行 = 最终一致（无锁）

自动后台化

getAutoBackgroundMs() = 120,000ms
超过 2 分钟的长任务将自动转为后台运行，避免阻塞父Agent。

终止方式

自然完成 / 父Agent调用 TaskStop / 达到 max_turns / 用户按下 Ctrl+C

Scratchpad 风险提示

文件读写无锁机制，并发写入时存在极低概率的数据丢失风险。作为Agent间唯一的共享状态，需谨慎使用。
